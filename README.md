# OVERWATCHER

1학년 2학기 나르샤

### About

- 외출, 현장체험 학습등을 나갈 때 학생들의 위치를 파악하여 빠르게 인원 체크를 할 수 있고

   공지 사항을 빠르게 학생들에게 신속하게 전달 할 수 있는 앱 어플리케이션을 만들었습니다.

### Stack

- React Native

- mobx

- expo

- native base

- React navigation
